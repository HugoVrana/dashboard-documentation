# Roles & Grants

All role and grant endpoints are on the Auth service (`http://localhost:8081`) and require a valid Bearer token.

---

## Roles

### Get all roles

```http
GET /api/v2/role/
Authorization: Bearer <access_token>
```

### Response `200 OK`

```json
[
  {
    "id": "507f1f77bcf86cd799439011",
    "name": "admin",
    "grants": [
      { "id": "507f1f77bcf86cd799439012", "name": "dashboard-invoices-read", "description": "Read invoice data" }
    ]
  }
]
```

---

### Get role by ID

```http
GET /api/v2/role/{id}
Authorization: Bearer <access_token>
```

---

### Create role

**Required Grant**: `dashboard-oauth-role-create`

```http
POST /api/v2/role/
Authorization: Bearer <access_token>
Content-Type: application/json
```

```json
{ "name": "accountant" }
```

Returns the created `RoleRead` object. `409` if a role with that name already exists.

---

### Update role

**Required Grant**: `dashboard-oauth-role-update`

```http
PUT /api/v2/role/{id}
Authorization: Bearer <access_token>
Content-Type: application/json
```

```json
{ "name": "senior-accountant" }
```

---

### Delete role

**Required Grant**: `dashboard-oauth-role-delete`

```http
DELETE /api/v2/role/{id}
Authorization: Bearer <access_token>
```

Returns `204 No Content`.

---

### Add grant to role

**Required Grant**: `dashboard-oauth-role-manage-grants`

```http
POST /api/v2/role/{id}/grants/{grantId}
Authorization: Bearer <access_token>
```

Returns the updated `RoleRead`. `409` if the grant is already assigned.

---

### Remove grant from role

**Required Grant**: `dashboard-oauth-role-manage-grants`

```http
DELETE /api/v2/role/{id}/grants/{grantId}
Authorization: Bearer <access_token>
```

Returns the updated `RoleRead`.

```typescript
  id: string;

```

## Standard Grants

### Dashboard API grants

| Grant | Description |
|-------|-------------|
| `dashboard-customers-read` | Read customer data |
| `dashboard-customers-create` | Create customers |
| `dashboard-customers-update` | Update customers and upload images |
| `dashboard-customers-delete` | Delete customers |
| `dashboard-invoices-read` | Read invoice data |
| `dashboard-invoices-create` | Create new invoices |
| `dashboard-invoices-update` | Update existing invoices |
| `dashboard-invoices-delete` | Delete invoices |
| `dashboard-revenue-read` | Read revenue data |

### Auth service admin grants

| Grant | Description |
|-------|-------------|
| `dashboard-oauth-user-update` | Update user details |
| `dashboard-oauth-user-delete` | Delete users |
| `dashboard-oauth-user-block` | Block and unblock users |
| `dashboard-oauth-user-resend-verification` | Resend verification email |
| `dashboard-oauth-user-reset-password` | Trigger password reset for a user |
| `dashboard-oauth-role-create` | Create roles |
| `dashboard-oauth-role-update` | Update roles |
| `dashboard-oauth-role-delete` | Delete roles |
| `dashboard-oauth-role-manage-grants` | Add/remove grants on roles |
| `dashboard-oauth-grant-create` | Create grants |
| `dashboard-oauth-grant-update` | Update grants |
| `dashboard-oauth-grant-delete` | Delete grants |
| `dashboard-oauth-client-create` | Register OAuth clients |
| `dashboard-oauth-client-rotate-secret` | Rotate an OAuth client secret |
| `dashboard-oauth-client-delete` | Delete OAuth clients |
