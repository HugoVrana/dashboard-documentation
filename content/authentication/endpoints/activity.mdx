# Activity Endpoints

Activity endpoints on the Auth service mirror those on the Dashboard API. Base URL: `http://localhost:8081`

## Get Recent Activity

```http
GET /api/v2/activity/recent?limit=50
```

### Query Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `limit` | integer | No | 50 | Maximum events to return |

### Response `200 OK`

```json
[
  {
    "id": "507f1f77bcf86cd799439011",
    "type": "USER_REGISTERED",
    "actorId": "507f1f77bcf86cd799439013",
    "actorName": "Jane Smith",
    "timestamp": "2025-01-01T08:00:00.000Z",
    "metadata": [
      { "key": "value" }
    ]
  }
]
```

## WebSocket

Real-time activity events are also available via WebSocket at `ws://localhost:8081/ws`.
