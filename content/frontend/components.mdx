# Components

## Base Components

Located in `app/ui/base/`, built on **Base UI** and **Radix UI** primitives.

### Installation

Install base components with shadcn wrapper:

```bash
npx shadcn@latest add <component-name>
```

### Available Components

| Component | File | Description |
|-----------|------|-------------|
| Button | `button.tsx` | Primary button with variants |
| Card | `card.tsx` | Container card |
| Input | `input.tsx` | Text input field |
| Label | `label.tsx` | Form label |
| Select | `select.tsx` | Dropdown select |
| Combobox | `combobox.tsx` | Searchable dropdown |
| Badge | `badge.tsx` | Status badges |
| Tabs | `tabs.tsx` | Tab navigation |
| AlertDialog | `alert-dialog.tsx` | Confirmation dialogs |
| DropdownMenu | `dropdown-menu.tsx` | Context menus |
| NavigationMenu | `navigation-menu.tsx` | Navigation |
| ScrollArea | `scroll-area.tsx` | Scrollable container |
| Separator | `separator.tsx` | Visual divider |
| Switch | `switch.tsx` | Toggle switch |
| Textarea | `textarea.tsx` | Multi-line input |
| Field | `field.tsx` | Form field wrapper |
| InputGroup | `input-group.tsx` | Input with addons |

## Component Patterns

### Server vs Client Components

```tsx
// Server Component (default)
export default async function ServerComponent() {
  const data = await fetchData();
  return <div>{data}</div>;
}

// Client Component
'use client';
export function ClientComponent() {
  const [state, setState] = useState();
  return <button onClick={() => setState()}>Click</button>;
}
```

### Form Actions

```tsx
// Server Action
async function createInvoice(formData: FormData) {
  'use server';
  // Handle form submission
}

// In component
<form action={createInvoice}>
  <input name="amount" type="number" />
  <button type="submit">Create</button>
</form>
```
