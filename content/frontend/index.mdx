# Frontend Overview

The frontend is built with **Next.js** using the App Router, **React 19**, **TypeScript**, and **Tailwind CSS**.

## Project Structure

```
app/
├── page.tsx                    # Home page
├── homeContent.tsx
├── layout.tsx                  # Root layout
├── auth/                       # Authentication pages
│   ├── login/
│   ├── roles/
│   ├── unauthorized/
│   ├── permission/
│   ├── components/             # Auth-specific components
│   ├── actions/                # Auth server actions
│   ├── dataAccess/
│   ├── hooks/
│   ├── models/
│   └── typeValidators/
├── dashboard/                  # Dashboard pages
│   ├── (overview)/             # /dashboard route group
│   ├── invoices/               # Invoice management pages
│   ├── components/             # Dashboard components
│   │   ├── dashboard/          # Cards, revenue chart, latest invoices
│   │   └── activity/           # Activity feed
│   ├── dataAccess/             # API clients
│   ├── models/
│   ├── hooks/
│   └── actions.ts              # Server actions
└── shared/                     # Shared across features
    ├── api/                    # Next.js API routes (Loki proxy)
    ├── components/             # Shared UI components
    │   ├── navigation/         # Navbar, language/theme toggles
    │   └── devOverlay/         # Dev debugging panel
    ├── dataAccess/             # Shared API clients (activity, Grafana)
    ├── contexts/               # Theme and translation contexts
    ├── models/
    └── i18n/
```

## Key Features

- **Server Components** - Default React Server Components with selective client components
- **App Router** - Next.js file-based routing with route groups
- **Type Safety** - Full TypeScript coverage
- **Theming** - Light/dark mode via context
- **i18n** - Internationalization via next-intl
- **WebSockets** - Real-time activity feed on the home page
- **Dev Overlay** - In-app panel for inspecting API context, translations, and user info

```env
GRAFANA_URL=http://localhost:3100
```

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
