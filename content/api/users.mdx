# Users API

Endpoints for user management in the Dashboard service.

## Get All Users

Retrieves all users.

**Required Grant**: `dashboard-users-read`

```http
GET /users/
```

### Response

```json
[
  {
    "id": "507f1f77bcf86cd799439011",
    "name": "John Doe",
    "email": "john@example.com"
  }
]
```

---

## Get User by ID

Retrieves a single user by their ID.

**Required Grant**: `dashboard-users-read`

```http
GET /users/{id}
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | 24-character hex ObjectId |

### Response

```json
{
  "id": "507f1f77bcf86cd799439011",
  "name": "John Doe",
  "email": "john@example.com"
}
```

### Errors

| Code | Description |
|------|-------------|
| `404` | User not found or invalid ID |

---

## Get User by Email

Retrieves a user by their email address.

**Required Grant**: `dashboard-users-read`

```http
GET /users/email/{email}
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `email` | string | Valid email address |

### Response

```json
{
  "id": "507f1f77bcf86cd799439011",
  "name": "John Doe",
  "email": "john@example.com"
}
```

---

## Search Users

Search and paginate through users.

**Required Grant**: `dashboard-users-create`

```http
POST /users/search
Content-Type: application/json
```

### Request Body

```json
{
  "page": 1,
  "size": 15,
  "search": "john"
}
```

### Response

```json
{
  "data": [
    {
      "id": "507f1f77bcf86cd799439011",
      "name": "John Doe",
      "email": "john@example.com"
    }
  ],
  "totalPages": 1,
  "currentPage": 1,
  "itemsPerPage": 15
}
```

Returns `204 No Content` if no users match the search.
