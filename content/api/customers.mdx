# Customers API

Endpoints for managing customer data.

## Get All Customers

**Required Grant**: `dashboard-customers-read`

Retrieves a list of all customers.

```http
GET /api/v1/customers/
```

### Response

```json
[
  {
    "id": "507f1f77bcf86cd799439011",
    "name": "John Doe",
    "email": "john@example.com",
    "image_url": "/customers/john-doe.png"
  }
]
```

### TypeScript Type

```typescript
type CustomerRead = {
  id: string;
  name: string;
  email: string;
  image_url: string;
}
```

---

## Get Customer by ID

Retrieves a single customer by their ID.

```http
GET /api/v1/customers/{id}
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | 24-character hex ObjectId |

### Response

```json
{
  "id": "507f1f77bcf86cd799439011",
  "name": "John Doe",
  "email": "john@example.com",
  "image_url": "/customers/john-doe.png"
}
```

### Errors

| Code | Description |
|------|-------------|
| `404` | Customer not found or invalid ID |

Returns the total number of customers.

```http
GET /api/v1/customers/count
```

## Create Customer

Creates a new customer.

**Required Grant**: `dashboard-customers-create`

```http
POST /api/v1/customers/
Content-Type: application/json
```

### Request Body

```json
{
  "name": "Jane Smith",
  "email": "jane@example.com"
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | Yes | Customer display name |
| `email` | string | Yes | Valid email address |

### Response

Returns `201 Created` with the created customer.

```json
{
  "id": "507f1f77bcf86cd799439011",
  "name": "Jane Smith",
  "email": "jane@example.com",
  "image_url": null
}
```

---

## Upload Customer Image

Uploads a profile image for a customer.

**Required Grant**: `dashboard-customers-update`

```http
POST /api/v1/customers/{id}/image
Content-Type: multipart/form-data
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | 24-character hex ObjectId |

### Form Data

| Field | Type | Description |
|-------|------|-------------|
| `file` | file | Image file to upload |

### Response

Returns a string with the public URL of the uploaded image.

```json
"https://storage.example.com/customers/507f1f77bcf86cd799439011.png"
```

---

## Update Customer

Updates an existing customer.

**Required Grant**: `dashboard-customers-update`

```http
PUT /api/v1/customers/{id}
Content-Type: application/json
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | 24-character hex ObjectId |

### Request Body

```json
{
  "id": "507f1f77bcf86cd799439011",
  "name": "Jane Smith Updated",
  "email": "jane.updated@example.com"
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `id` | string | Yes | Customer ObjectId |
| `name` | string | Yes | Customer display name |
| `email` | string | Yes | Valid email address |
| `imageId` | string | No | ObjectId of uploaded image |

### Response

Returns the updated `CustomerRead` object.

---

## Delete Customer

Soft-deletes a customer by setting the `deletedAt` audit field.

**Required Grant**: `dashboard-customers-delete`

```http
DELETE /api/v1/customers/{id}
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | 24-character hex ObjectId |

### Response

```json
1
```

Returns `1` if successfully deleted.
