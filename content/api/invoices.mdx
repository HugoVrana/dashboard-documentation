# Invoices API

Endpoints for managing invoice data. Invoices track payments from customers.

## Get All Invoices

Retrieves all invoices with customer details.

**Required Grant**: `dashboard-invoices-read`

```http
GET /invoices/
```

### Response

```json
[
  {
    "id": "507f1f77bcf86cd799439011",
    "status": "pending",
    "amount": 1500.00,
    "date": "2024-01-15",
    "customer": {
      "id": "507f1f77bcf86cd799439012",
      "name": "John Doe",
      "email": "john@example.com",
      "image_url": "/customers/john-doe.png"
    }
  }
]
```

---

## Get Invoice by ID

Retrieves a single invoice by ID.

**Required Grant**: `dashboard-invoices-read`

```http
GET /invoices/{id}
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | 24-character hex ObjectId |

---

## Get Latest Invoices

Retrieves the most recent invoices, optionally within a range.

**Required Grant**: `dashboard-invoices-read`

```http
GET /invoices/latest?indexFrom=0&indexTo=5
```

### Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `indexFrom` | integer | No | Start index (inclusive) |
| `indexTo` | integer | No | End index (inclusive) |

---

## Get Invoice Count

Returns the count of invoices, optionally filtered by status.

**Required Grant**: `dashboard-invoices-read`

```http
GET /invoices/count?status=pending
```

### Query Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `status` | string | No | Filter by status: `pending` or `paid` |

### Response

```json
15
```

## Get Invoice Pages

Returns the total number of pages for pagination.

**Required Grant**: `dashboard-invoices-read`

```http
GET /invoices/pages?searchTerm=john&size=15
```

### Query Parameters

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `searchTerm` | string | No | - | Search term |
| `size` | integer | No | 15 | Items per page |

## Search Invoices

Search and paginate through invoices.

**Required Grant**: `dashboard-invoices-read`

```http
POST /invoices/search
Content-Type: application/json
```

### Request Body

```json
{
  "page": 1,
  "size": 15,
  "search": "john"
}
```

### Response

```json
{
  "data": [
    {
      "id": "507f1f77bcf86cd799439011",
      "status": "pending",
      "amount": 1500.00,
      "date": "2024-01-15",
      "customer": {
        "id": "507f1f77bcf86cd799439012",
        "name": "John Doe",
        "email": "john@example.com",
        "image_url": "/customers/john-doe.png"
      }
    }
  ],
  "totalPages": 5,
  "currentPage": 1,
  "itemsPerPage": 15
}
```

## Create Invoice

Creates a new invoice.

**Required Grant**: `dashboard-invoices-create`

```http
POST /invoices/
Content-Type: application/json
```

### Request Body

```json
{
  "status": "pending",
  "amount": 1500.00,
  "customer_id": "507f1f77bcf86cd799439012"
}
```

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `status` | string | Yes | `pending` or `paid` |
| `amount` | number | Yes | Invoice amount |
| `customer_id` | string | Yes | 24-char hex ObjectId of customer |

### Response

Returns `201 Created` with the created invoice and `Location` header.

```json
{
  "id": "507f1f77bcf86cd799439011",
  "status": "pending",
  "amount": 1500.00,
  "date": "2024-01-15",
  "customer": {
    "id": "507f1f77bcf86cd799439012",
    "name": "John Doe",
    "email": "john@example.com",
    "image_url": "/customers/john-doe.png"
  }
}
```

## Delete Invoice

Soft-deletes an invoice by setting the `deletedAt` audit field.

**Required Grant**: `dashboard-invoices-delete`

```http
DELETE /invoices/{id}
```

### Path Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `id` | string | 24-character hex ObjectId |

### Response

```json
1
```

Returns `1` if successfully deleted.
