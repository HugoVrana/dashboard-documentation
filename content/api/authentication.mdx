# Authentication

Learn how to authenticate with the Dashboard API.

## API Keys

Generate an API key from your dashboard settings.

## Usage

Include your API key in the request header:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.yourdomain.com/v1/endpoint
```

## Token Expiration

API tokens expire after 30 days. Refresh tokens before expiration to maintain access.
