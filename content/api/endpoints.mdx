# API Endpoints

## Dashboard

### Get Dashboard

```http
GET /dashboard
```

Returns the current dashboard configuration.

### Update Dashboard

```http
PUT /dashboard
```

Updates the dashboard configuration.

## Widgets

### List Widgets

```http
GET /widgets
```

Returns a list of all available widgets.

### Create Widget

```http
POST /widgets
```

Creates a new widget.

### Delete Widget

```http
DELETE /widgets/:id
```

Deletes a widget by ID.
