## Technology Stack

| Component | Technology | Purpose |
|-----------|------------|---------|
| Backend | Spring WebSocket | WebSocket support in Spring |
| Backend | STOMP | Simple messaging protocol |
| Frontend | SockJS | WebSocket with fallback support |
| Frontend | STOMP.js | STOMP client for JavaScript |

## Key Concepts

### STOMP Protocol

STOMP (Simple Text Oriented Messaging Protocol) provides a frame-based protocol over WebSocket:

- **CONNECT** - Establish connection
- **SUBSCRIBE** - Subscribe to a destination (topic)
- **SEND** - Send message to a destination
- **MESSAGE** - Server-to-client message
- **DISCONNECT** - Close connection

### SockJS Fallback

SockJS provides WebSocket emulation when native WebSocket is unavailable:

1. **WebSocket** - Native protocol (preferred)
2. **XHR Streaming** - Long-polling fallback
3. **XHR Polling** - Short-polling fallback
