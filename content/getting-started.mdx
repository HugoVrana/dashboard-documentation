# Getting Started

Learn how to set up and configure the Dashboard application.

## Prerequisites

Before you begin, ensure you have the following:

- Node.js 18 or later
- npm or yarn package manager

## Installation

```bash
npm install
```

## Configuration

Configure your dashboard by setting up the required environment variables.

## Running the Application

```bash
npm run dev
```

Your dashboard will be available at `http://localhost:3000`.
