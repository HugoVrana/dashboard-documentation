# Architecture Overview

The Dashboard application follows a microservices architecture with clear separation between frontend, backend APIs, and authentication services. Real-time updates are delivered via WebSockets.

## Project Repositories

| Repository | Description | Technology |
|------------|-------------|------------|
| `nextjs-dashboard-frontend` | Frontend application | Next.js 15, React 19 |
| `spring-dashboard` | Main backend API | Spring Boot, MongoDB |
| `spring-dashboard-common` | Shared library | Java |
| `spring-dashboard-oauth` | Authentication service | Spring Boot, JWT |
